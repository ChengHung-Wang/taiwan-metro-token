# 台灣捷運票卡資料集 (Taiwan Metro Token)

![project-banner](img/banner-with-ticket.png)


## 專案介紹

這個倉庫收集了台北捷運、桃園機場捷運、台中捷運與高雄捷運等系統的各種非接觸式票卡資料，包括單程票、一日票、旅遊票與紀念票等。所有票卡經由 Flipper Zero 或 Proxmark3 裝置讀取後提供 `.nfc`、`.bin`、`.json` 等檔案，方便研究台灣票卡系統的運作與逆向工程。資料僅供研究之用，請勿用於實際乘車或其他商業用途。

歡迎對台灣票卡生態、逆向工程有興趣的研究者與開發者參考、分析或貢獻此資料集。

## dump 檔說明

| 格式 | 來源裝置 | 說明 |
|------|-----------|------|
| `.nfc` | Flipper Zero | FeliCa 卡片資料，由 Flipper Zero 讀取後儲存的票卡資訊，內容包含 UID、區塊資料及其他屬性 |
| `.bin` | Proxmark3 | 由 Proxmark3 讀取的二進制檔案( `hf mfu dump` ) |
| `.json` | Proxmark3 | 對應 `.bin` 檔的解析版本，列出卡片 UID、版本資訊以及各記憶體區塊的十六進位值 |

### 刑法

**第 203 條**<br>
意圖供行使之用，而偽造、變造船票、火車、電車票或其他往來客票者，處一年以下有期徒刑、拘役或九千元以下罰金；其行使之者，亦同。

**第 205 條（沒收物）**<br>
偽造、變造之有價證券、郵票、印花稅票、信用卡、金融卡、儲值卡或其他相類作為提款、簽帳、轉帳或支付工具之電磁紀錄物及前條之器械原料及電磁紀錄，<ins>不問屬於犯人與否，沒收之</ins>。

**第 210 條（偽造變造私文書罪）**<br>
偽造、變造私文書，足以生損害於公眾或他人者，處五年以下有期徒刑。

**第 339‑1 條（違法由收費設備取得他人之物之處罰）**<br>
一、意圖為自己或第三人不法之所有，以不正方法由收費設備取得他人之物者，處一年以下有期徒刑、拘役或十萬元以下罰金。<br>
二、以前項方法得財產上不法之利益或使第三人得之者，亦同。<br>
三、前二項之<ins>未遂犯罰之</ins>。

> 收費設備: 票卡機、閘門<br>
> 取得他人之物: 包括取得服務（你透過捷運從A站搭到B站）。

**第 339-3 條（違法製作財產權紀錄取得他人之物之處罰）**<br>
一、意圖為自己或第三人不法之所有，以不正方法將虛偽資料或不正指令輸入電腦或其相關設備，製作財產權之得喪、<ins>變更紀錄</ins>，而取得他人之財產者，<ins>處七年以下有期徒刑，得併科七十萬元以下罰金</ins>。<br>
二、以前項方法得財產上不法之利益或使第三人得之者，亦同。<br>
三、前二項之未遂犯罰之。<br>

> 實務上曾認為: 只要造成後端系統產生未曾實際有的「額度增減」紀錄，則等於以虛偽資料製作財產權變更紀錄，即適用刑 339-3 條([高中生刷退 TPass 2.0 案](https://www.chinatimes.com/realtimenews/20251231001729-260402)，士林地院 114 年審訴字第 2162 號)。 注意，這裡不論有沒有成功得到利益（未遂與否），只要有辦法讓後端系統製作虛偽財產變更紀錄就算。

### 大眾捷運法(交通部法規)

**第 49 條 – 無票、失效票或冒用他人票券之罰則**

旅客<ins>無票</ins>、持用失效車票或冒用不符身分之車票乘車者，除補繳票價外，並<ins>**支付票價五十倍**</ins>之違約金。前項應補繳票價及支付之違約金，如旅客不能證明其起站地點者，以營運機構公告之單程票最高票價計算。

> ##### 關於無票的定義
> - 臺北捷運系統旅客須知 - 「無票」：指旅客未持有效車票或持用經變造、偽造之車票進入付費區或乘車者。
> - 新北大眾捷運股份有限公司旅客須知（中運量捷運系統）- 「無票」：指旅客未持有效車票或持用經變造、偽造或不符使用身分之車票進入付費區或乘車
> - 臺中捷運股份有限公司旅客須知 - 「無票乘車」：指旅客未持有效車票或持用經由變造、偽造之車票進入付費區
> - 高雄捷運旅客須知 - 「無票」：未持有車票或持用經變造、偽造或不符使用身分之車票進出付費區或乘車者

### 鐵路法(交通部法規)

**第 65 條**<br>
車票加價出售或換取不正利益圖利者，按車票張數，處其運價之十倍至五十倍罰鍰；加價出售取票憑證圖利者，亦同。<ins>以不正方法將虛偽資料或不正指令輸入電腦或其相關設備而購買車票、取得訂票或取票憑證者，處五年以下有期徒刑或科或併科新臺幣三百萬元以下罰金</ins>。
